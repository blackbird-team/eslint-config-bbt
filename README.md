# eslint-config-bbt
ESLint Config for JavaScript used by BlackBird Team

## Install
```cmd
npm install --save-dev eslint eslint-config-bbt
```

```cmd
npm i -D eslint-config-bbt
```

## Usage
```eslint
{
	...
	"extends": [ "bbt" ]
	...
}
```
